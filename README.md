-   [Nodejs](https://nodejs.org/en) : 20.24.0 <br>
-   [TailwindCSS](https://tailwindcss.com/) : 3.4.4 <br>
-   [Redux](https://redux.js.org/) : 5.0.1 <br>
-   [Redux Toolkit](https://redux-toolkit.js.org/) : 2.2.6 <br>
-   [ESLint](https://eslint.org/) : 8.57.0 <br>

## How to Start Project is using npm

```bash
git clone https://github.com/alfazh123/FrontendDevReactjs-AhmdMufahrasLiAlfazhAssardew.git
```

```bash
cd FrontendDevReactjs-AhmdMufahrasLiAlfazhAssardew/
```

## Setup

```bash
npm install
```

## Run Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
