# Getting Started

This project was built with evirontment as below :

Reactjs Version : 18.3.1
nodejs version : 20.24.0
Tailwind css : 3.4.4
redux : 5.0.1
redux/toolkit : 2.2.6
eslint : 8.57.0

How to Start Project is using npm

```js
// you can clone from this repo first

git clone https://github.com/alfazh123/FrontendDevReactjs-AhmdMufahrasLiAlfazhAssardew.git
```

After cloning the repo, do the following steps below.

```bash
cd FrontendDevReactjs-AhmdMufahrasLiAlfazhAssardew/
```

Next, install node_module

```bash
npm install
```

Then run the project by running the command below

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
